# 请求与反馈

> 您的每一条观影期待将在获取后第一时间更新；如因资源稀缺或其他客观因素导致请求失败，我们深感抱歉

## **一、影片请求**

### 1. 前往[**媒体支持与服务中心**](https://mediasupport.dimlight.top/)搜索或选择需要请求的影片

![Jellyseerr-home.webp](/Jellyseerr-home.webp)

### 2. 点击提交请求

![Jellyseerr-request.webp](/Jellyseerr-request.webp)

### 3. 选择需要请求的季度并确认

![Jellyseerr-request-information.webp](/Jellyseerr-request-information.webp)

## **二、反馈问题**

### 1. 前往[**媒体支持与服务中心**](https://mediasupport.dimlight.top/)搜索或选择需要反馈问题的影片

![Jellyseerr-home.webp](/Jellyseerr-home.webp)

### 2. 点击报告问题

![Jellyseerr-feedback.webp](/Jellyseerr-feedback.webp)

### 3. 填写报告问题的详细信息并提交

![Jellyseerr-feedback-information.webp](/Jellyseerr-feedback-information.webp)
